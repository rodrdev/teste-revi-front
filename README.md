## Requisitos
- Node.js >= 18.x
- npm >= 9.x ou yarn >= 1.22.x

## Instalação

1. Clone o repositório:
```bash
git clone https://github.com/rodrdev/teste-revi-front.git

cd teste-revi-front
```

2. Instale as dependências:
```bash
npm install
# ou
yarn
```
3. Como rodar:
```bash
npm run dev
# ou
yarn dev

A aplicação estará disponível em: http://localhost:5173
```
